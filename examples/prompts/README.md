# Prompt Examples

These prompts show how to use the rules with AI coding tools.

They are intentionally framework-neutral. Add your project stack, file paths, and design system details when using them in a real repository.

- [Generate User Management List](./generate-user-management-list.md)
- [Generate Dialog Form](./generate-dialog-form.md)

## Usage Pattern

1. Pick the closest prompt.
2. Replace the scenario details.
3. Add the relevant rule bundle paths.
4. Add your technology stack and existing component library.
5. Ask AI to generate or review against the rules.
