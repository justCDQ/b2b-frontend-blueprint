## Use For

- Console shell.
- Sidebar/topbar navigation.
- Settings or admin module structure.
- Page header and breadcrumb/back behavior.
- Route-backed tabs and detail sub-sections.

## Must Enforce

- IA follows user tasks/resource domains.
- Every route is reachable or redirected.
- Page title exists for routed pages.
- Breadcrumb and Back have distinct roles.
- Active route remains visible.
- Query params are used only for recoverable page state.
- Responsive navigation keeps active route and primary actions reachable.
